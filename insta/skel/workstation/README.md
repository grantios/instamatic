# workstation Configuration Skeleton

This directory contains configuration files for the default workstation setup.

## Directory Structure:
- **@sys/** - Files copied to system root (/)
  - **etc/issue** - Login banner with GrantiOS branding and post-install instructions

## Usage:
Place configuration files in the appropriate directories above.
They will be automatically copied during workstation installation.

## Notes:
This skeleton is used by the default workstation.sh configuration.
